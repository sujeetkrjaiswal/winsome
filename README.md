# winsome
Design System

## commit messages

### General Pattern for comment
```text
type(scope?): subject
```

* **scope is optional**
* **multiple scopes are supported (current delimiter options: "/", "\" and ",")**

#### Example 1

```text
chore: run tests on travis ci
```

#### Example 2

```text
fix(server): send cors headers
```

#### Example 2

```text
feat(blog): add comment section
```
